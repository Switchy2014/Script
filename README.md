# Script! Build, Run, Share!

A visual block-based IDE that compiles to JavaScript and runs inside a sandboxed runtime.

## Features
- Blockly-based visual programming
- JavaScript code generation
- Sandbox execution
- Canvas-based output stage
- Start / Stop execution system

## Goal
Make programming more visual, while still generating real JavaScript.
